## The Strategic "Why" (Overview)

> <blockquote>
>   Manual grade calculation is often prone to human error, incredibly time-consuming for educators, and frequently lacks a standardized approach. This leads to inconsistencies in student assessment, increased administrative overhead, and potential inaccuracies that can impact student academic records.
> </blockquote>

The `week2-grade-calculator` project delivers a robust, automated Python solution designed to accurately compute student grades. By automating the assessment process, it not only significantly reduces the time and effort spent on manual calculations but also ensures a standardized, error-free approach to grading. This project empowers educators with a reliable tool to streamline their workflow, providing consistent and trustworthy results for every student.

## Operational Setup

### Prerequisites

Ensure you have the following runtime installed on your system:

*   **Python 3.x**: This project is built and tested with Python 3.

### Installation

Follow these steps to get the `week2-grade-calculator` up and running on your local machine:

```bash
# 1. Clone the repository
git clone https://github.com/your-username/week2-grade-calculator.git
cd week2-grade-calculator

# 2. Run the grade calculator script
#    Ensure 'test_students.txt' or your desired student data file is in the same directory.
python grade_calculator.py test_students.txt
```
