# Student Grade Calculator

## Project Description
A comprehensive Python application that processes multiple students, calculates grades, generates comments, provides statistics, supports searching students, and saves results to a file.

## Features
- Grade Calculation
- Input Validation
- Error Handling
- Lists & Loops
- Student Search
- Save Results to File
- Class Statistics

## How to Run
python grade_calculator.py
